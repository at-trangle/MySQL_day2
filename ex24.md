24. Lấy 1 blog và 1 news có số lượng comment nhiều nhất
```
(
    select title
    from blog inner join comment on blog.id = comment.target_id
    where target_table = 'blog'
    group by target_id
    order by count(target_id) desc
    limit 1
)
union all
(
    select title
    from news inner join comment on news.id = comment.target_id
    where target_table = 'news'
    group by target_id
    order by count(target_id) desc
    limit 1
);
```
