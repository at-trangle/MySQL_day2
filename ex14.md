14. Lấy 3 User comment đầu tiên trong 5 blogs mới nhất
```

select distinct full_name from user inner join
         (select comment.user_id from comment inner join 
                      (select id from blog order by created_at desc limit 5) as new5Blog 
          on comment.target_id = new5Blog.id
          where target_table = 'blog'
          order by created_at asc limit 3) as newComment
on user.id = newComment.user_id;
```
